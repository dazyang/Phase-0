Technical blog 4 — JavaScript Basics

Discuss the following question to a lay person:

  • How does Javascript compare to HTML and CSS?

    • While HTML and CSS makes the structural and visual presentation of the website, JavaScript is a programming language that is used to build interactive elements and animation to the webpage.

  • Explain control flow and loops using an example process from everyday life, for example 'walking up' or 'brushing your teeth'.

    • Control flow statement are used to determine what block of code to run, or not run, at a given time. There are two types of control flow: conditional statements and loops. Condition statements are a type of flow that determine when and where to execute code based on specified conditions. In JavaScript, we use if/else statements (or a switch statement) to apply conditions. To put it in an everyday life example: Loops on the other hand, are a block of code conditions we create to examine over and over again until the condition applies.

  • Explain the difference between accessing data from arrays and objects.

    • There are two ways to access object properties:
    objectName.propertyName || objectName[ 'propertyName' ]
    To access an array you refer to an array element by referring to its index number, for example:
    var name = whale[ 0 ];

  • Explain what functions are and why they are useful.

    • Functions in JavaScript are a block of reusable codes that can be used to perform repetitive tasks. Functions is useful because instead of writing the same code repeatedly to perform the same task, we can put it into a function and later call the name of the function to use.
