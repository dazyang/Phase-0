Technical blog 6 — JavaScript Language

Explain to a non-tech friend

  • In JavaScript Language, describe all the use cases for each of:

    • parentheses ( ) – Execute a function call, eg. console.log('The value between this parentheses will print out in console.')

    Square brackets [ ] – Square brackets are used to store sequences of values that called arrays. Eg. var list = [ 'item1', 'item2', 'item3' ]

    Curly braces { } – Curly braces are used for grouping statements together into a single statement that creates a block of code to be executed.

    single ' ' & double " " – JavaScript makes a string value out of everything marked in quotes. (Strings are used to represent text.) See the previous parentheses( ) example.
